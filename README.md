# BlueKit

Bluetooth hacking/packet analysis tool kit

OS requirements:  
* Debian or Ubuntu
* Bluetooth utils
* Python3
* Tshark

Pip requirements:  
* Pybluez 
* Keyboard
* Pyshark

### Set up  (RAN USING SUDO)
```bash
apt update
apt install tshark
sudo apt-get install python-dev
sudo apt-get install bluetooth libbluetooth-dev
pip install pybluez
pip install keyboard
pip install pyshark
```

### Usage:
```bash
sudo python3 bluekit.py
```
