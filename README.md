# BlueKit

Bluetooth hacking/packet analysis tool kit

OS requirements:  
* Debian or Ubuntu
* Pybluez tools (eg. bluetoothctl)
* Python3
* Tshark

Pip requirements:  
* Pybluez 
* Keyboard
* Pyshark
